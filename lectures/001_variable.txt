Variable
    เป็นชื่อของข้อมูลที่เราตั้ง เพื่อสำหรับเข้าถึงข้อมูลบน memory
    โดยรูปแบบการประกาศมีดังนี้

        type name = value;

    Data type 
        - int สำหรับจัดเก็บเลขจำนวนเต็ม ทั้งจำนวนลบและบวก ไม่มีทศนิยม
        - float ใช้จัดเก็บเลขจำนวนเต็มที่มีทศนิยม
        - double ใช้จัดเก็บเลขจำนวนเต็มที่มีทศนิยม แต่สามารถเก็บได้เยอะกว่า float
        - char ใช้ำหรับจัดเก็บตัวอักษร
        - char[] เป็นกลุ่มของตัวอักษร (string)

    Data type unsigned
        เป็นประเภทข้อมูลเกี่ยวกับตัวเลขที่จะเก็บเฉพาะค่าที่เป้นบวกเท่านั้น

    printf()
        ใช้สำหรับแสดงข้อมูลที่เป็น string ออกไปที่ console
        โดยจะมี format ของการแสดงข้อมูลดังนี้
            - %d ใช้สำหรับแสดงเลขที่ไม่มีทศนิยม
            - %f ใช้สำหรับแสดงเลขที่มีทศนิยม
            - %lf ใช้จัดเก็บเลขจำนวนเต็มที่มีทศนิยมที่มีจำนวนแบบ double
            - %c ใช้จัดเก็บเลขตัวอักษร
