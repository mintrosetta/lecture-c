Scanf
    เป็นฟังก์ชันพื้นฐานในภาษา C ใช้สำหรับการรับข้อมูลมาจาก terminal
    แต่เราสามารถสั่งให้รับข้อมูลประเภทอื่น ๆ ได้ เช่น ไฟล์

* scanf(ประเภทของข้อมูลที่คาดหวัง, จะให้เก็บไว้ที่ไหน โดยจะต้องระบุเป็น pointer);
* scanf("%d", &n);
* &n หมายถึง ดึงที่อยู่ใน memory ของตัวแปร n

ประเภทของข้อมูล
    - %d    รับข้อมูลในรูปแบบ integer
    - %c    รับข้อมูลในรูปแบบ character
    - %f    รับข้อมูลในรูปแบบ float
    - %lf   รับข้อมูลในรูปแบบ double
