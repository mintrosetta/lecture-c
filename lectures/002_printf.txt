Printf
    เป็นฟังก์ชันมาตรฐานสำหรับแสดงนื้หาไปที่ terminal

Special Character
    - \n สำหรับขึ้นบรรทัดใหม่
    - \t เป็นการเว้นช่องว่าง 1 tap
    - \" แสดงผล double quote

Format 
    ใช้สำหรับแสดงข้อมูลที่เป็น string ออกไปที่ console
    โดยจะมี format ของการแสดงข้อมูลดังนี้
        - %d ใช้สำหรับแสดงเลขที่ไม่มีทศนิยม
        - %f ใช้สำหรับแสดงเลขที่มีทศนิยม
        - %lf ใช้จัดเก็บเลขจำนวนเต็มที่มีทศนิยมที่มีจำนวนแบบ double
        - %c ใช้จัดเก็บเลขตัวอักษร
        - %s ใช้จัดเก็บข้อมูลที่เป็น string

    รูปแบบของ format ทดังนี้
    %[flages][width][.percision]specifier
        - flages เป็นการระบุจำนวนช่องว่างก่อนการแสดงผลข้อมูล
        - width
        - percision เป็นการระบุจำนวนทศนิยมที่ต้องการ
        - specifier เป็นการระบุประเภทของข้อมูล เช่น d, c, f
